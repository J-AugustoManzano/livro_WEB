Início - Introdução

O termo programação é aqui aplicado no sentido de representar a atividade denominada programação de computadores, a qual é descrita no dicionário Aurélio como sendo a ciência ou técnica de elaboração de programas de computador.

Início - Programação Computacional

A programação computacional usada na atualidade moderna é descrita sob dois aspectos: o aspecto da programação estruturada, onde um programa é deswenvolvido de forma modular sub-rotinas) e hierárquico (normalmente baseado-se na técnica top-down) buscando independêcia funcional de cada módulo e melhor legibilidade do código fonte; e o aspecto da programação orientada a objetos, onde os dados e rotinas (métodos) são tratados como um pacote único (objetos).

Início - Necessidade na Programação

Para se programar um computador é inicialmente necessário escolher uma linguagem de programação e gerar um programa fonte segundo a linguagem a ser usada. Normalmente depois de escrever o programa fonte é necessário transformá-lo em programa executável. e isso é realizado com o uso de um compialdor. Mas além dos compiladores exitem também os tradutores, os interpretadores e os pseudo-compiladores discutidos neste sites.
