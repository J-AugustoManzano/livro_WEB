Linguagem de Programação FORTRAN

A linguagem FORTRAN possui uma característica fundamental que jamais nenhuma outra linguagem de programação de computadores terá, por ser a primeira linguagem de alto nível desenvolvida. Foi a partir dela que surgiram outras; portanto, FORTRAN é considerada a raiz que deu origem à árvore das linguagens de programação de alto nível. O nome FORTRAN é uma abreviação de FORmula TRANslation, que significa tradutor de fórmulas. Essa linguagem possui a característica de ser uma ferramenta para facilitar a utilização de algoritmos e equações matemáticas em computadores eletrônicos, tendo como alvo as áreas técnica e científica. Por: Suely Toledo, Sandra Toledo e Augusto Manzano.
