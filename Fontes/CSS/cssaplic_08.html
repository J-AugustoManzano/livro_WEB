Lista de Linguagens Computacionais

A linguagem FORTRAN possui uma característica fundamental que jamais nenhuma outra linguagem de programação de computadores terá, por ser a primeira linguagem de alto nível desenvolvida. Foi a partir dela que surgiram outras; portanto, FORTRAN é considerada a raiz que deu origem à árvore das linguagens de programação de alto nível.

  1. COBOL
  2. BASIC
  3. Pascal
  4. C++
